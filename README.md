# LTW Renderer Plugin for Zalith Launcher
## About the Open Source License
This project is developed based on [ZalithLauncher/RendererPlugin](https://github.com/ZalithLauncher/RendererPlugin) and is open-sourced under the GPLv3 license. The GPLv3 license explicitly stipulates that any project modified based on GPLv3-licensed code must also be open-sourced under the GPLv3 license.
## Installation Method
Go to [Releases](https://github.com/csf0304/LTWRenderer/releases) and download the version suitable for your device.
After installation is complete, open Zalith Launcher, and you will see the newly installed renderer.
## Modifying the Source Code
### 1. Clone this project
``` bash
git clone https://github.com/csf0304/LTWRenderer.git
```
### 2. Enter the project directory
``` bash
cd LTWRenderer/
```
### 3. Build the code
If you have finished making modifications or simply want to test, use the following command to build:
``` bash
./gradlew build
```
